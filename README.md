# Softball Lineup App

A web app for planning a softball game lineup.

## What It Does

- Manage your roster (add and remove players)
- Build and reorder batting order
- Assign defensive positions for a 10-player defensive set
- Auto-fill defense from batting order
- Detect duplicate defensive assignments

## Tech Stack

- React
- TypeScript
- Vite

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
```

## Project Scripts

- `npm run dev`: Start development server
- `npm run build`: Type-check and build production bundle
- `npm run preview`: Preview production build locally
